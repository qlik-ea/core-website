    Terminology

    This pages contains terminology used around elastic, docker, orchestration, and deployment to get a common understanding used among teams, partners and customers.

    D

    Docker swarm - Docker native clustering system

    E

    Elasticsearch - Open Source, Distributed, RESTful Search Engine

    Enigma - Framework used for communicating with qlik engine.

    H

    Halyard - Library used for loading data into qlik engine.

    K

    Kibana - Browser based analytics search dashboard for Elasticsearch

    Kubernetes - Automated container deployment, scaling, and management

    L

    Logstash - Processes and transforms data, and sends it to e.g. Elasticsearch.

    M

    Marathon - A container orchestration platform for Mesos.

    Mira - A QIX Engine strategy service for Qlik Elastic.

    N

    Nginx - HTTP and reverse proxy server, a mail proxy server, and a generic TCP/UDP proxy server

    T

    Traefik - HTTP reverse proxy and load balancer.
